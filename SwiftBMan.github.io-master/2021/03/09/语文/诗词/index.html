诗词短语

《沁园春 · 长沙》毛泽东

漫江：满江。漫，满、遍。例：漫江碧透、漫山红艳

橘子洲：风景名胜，在长沙西面的湘江中。例：橘子洲头

舸：大船。例：百舸争流

翔：原指鸟盘旋而飞，可形容鱼在清澈的水中轻快地游动。例：鱼翔浅底

万类：指一切生物。例：万类复苏

霜天：指深秋。例：万类霜天竞自由

怅寥廓：面对广阔的宇宙惆怅感慨。

沉浮：比喻事物的盛衰、消长。

峥嵘：山势高峻，比喻超越寻常，不平常。

恰：正当，正好。

书生：读书人，可指青年学生。

意气：意志和气概。

挥斥：奔放。

遒：强劲有力。

激扬：激浊扬清。

粪土：把……看作粪土

万户侯：汉代侯爵的最高一级，享有万户农民的赋税，后泛指有高爵显位的人。

击水：可指游泳

遏：阻止
